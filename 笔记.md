## 备注：.env
```js
git clone代码后，会有默认的.env_sample文件
直接 "npm run start" 启动项目，会找不到.env文件
所以，在Mac系统，
// Create a .env file by copying the sample
cp .env_sample .env
但在windows系统下，
改一下名称即可，直接将.env_sample改成.env。
然后，就能顺利开启项目
```
